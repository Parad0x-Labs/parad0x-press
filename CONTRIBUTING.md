# Contributing to Parad0x Press

Thank you for your interest in contributing to Parad0x Press.

## Development Setup

1. Clone the repository
2. Run `npm install`
3. Connect an Android device or start an emulator
4. Run `npx react-native run-android`

## Code Style

- Use TypeScript for all React Native code
- Use Kotlin for Android native modules
- Follow existing code formatting conventions

## Pull Requests

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Reporting Issues

Please use GitHub Issues to report bugs or request features.
